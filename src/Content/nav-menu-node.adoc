=== Nav Menu Node

The nav menu node is the top level node that shows in the navigation that allows the editor to build out one of the following types:

. <<nav-group, Nav Group>>
. <<nav-list, Nav List>>
. <<nav-list-item, Nav list Item>>

*Schema* - nav-menu-node.json
